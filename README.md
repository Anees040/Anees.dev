# Anees.dev — Personal Portfolio

A fast, hand-crafted developer portfolio for **Muhammad Anees** — Flutter &amp; Mobile App Developer.
Built with plain **HTML, CSS &amp; vanilla JavaScript** (zero build step, zero dependencies) so it loads
instantly and deploys anywhere.

## ✨ Features
- Custom animated cursor + magnetic buttons
- Scroll-reveal animations (IntersectionObserver)
- Typewriter role headline
- Animated counters, gradient blobs & grid background
- 3D phone mockup with tilt-on-hover (SportLynk preview)
- Interactive skill cards with spotlight hover
- Case-study project sections + “coming soon” AI projects
- Journey timeline, contact section, fully responsive + mobile menu
- Respects `prefers-reduced-motion`

## 📁 Structure
```
.
├─ index.html      # markup / content
├─ css/style.css   # all styles
├─ js/main.js      # all interactions
└─ README.md
```

## 🚀 Run locally
Just open `index.html` in a browser — that's it.
Or run a tiny local server:
```bash
# Python
python3 -m http.server 5173
# then visit http://localhost:5173
```

## 🌐 Deploy (free)
**Vercel:** import the repo at vercel.com → New Project → select `Anees.dev` → Deploy (no settings needed).
**GitHub Pages:** repo Settings → Pages → Source: `main` / root → Save.

## 🛠 Customize
- Text/content → `index.html`
- Colors → CSS variables at the top of `css/style.css` (`--accent`, `--bg`, etc.)
- Add a real photo → replace the `.about__portrait` initials block with an `<img>`
